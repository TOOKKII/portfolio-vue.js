# Portfolio de Yasser BOURAGHDA

Bienvenue sur mon portfolio personnel ! 
Ce projet est une vitrine de mes compétences en développement web, conçue comme une **Single Page Application (SPA)** moderne et réactive.

## À propos du projet
L'objectif de ce portfolio est de présenter mes projets et mon parcours de manière fluide et interactive. J'ai choisi de construire cette application "from scratch" pour démontrer ma maîtrise de l'écosystème Vue.js et de l'intégration moderne.

## :tools: Stack Technique

Ce projet utilise les dernières technologies du web :

* **Framework :** [Vue.js 3](https://vuejs.org/) (Composition API & `<script setup>`)
* **Build Tool :** [Vite](https://vitejs.dev/) (pour des performances ultra-rapides)
* **Styling :** [Tailwind CSS](https://tailwindcss.com/) (Design responsive et utilitaire)
* **Langage :** JavaScript (ES6+)

## Fonctionnalités

* **Design Responsive :** S'adapte parfaitement aux mobiles, tablettes et ordinateurs.
* **Navigation Rapide :** Grâce à Vite et au Virtual DOM de Vue.
* **Section Contact :** Formulaire intégré pour me joindre facilement.

## Installation & Lancement

Si vous souhaitez voir le code tourner sur votre machine :

```bash
# 1. Cloner le projet
git clone [https://github.com/TOOKKII/my-project-vue3.git](https://github.com/TOOKKII/my-project-vue3.git)

# 2. Entrer dans le dossier
cd my-project-vue3

# 3. Installer les dépendances
npm install

# 4. Lancer le serveur de développement
npm run dev
